# Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in the **Hyper Learning Tech** community a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

We pledge to act and interact in ways that contribute to an open, welcoming, diverse, inclusive, and healthy community.

---

# Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and welcoming to everyone.
- Using inclusive and professional language.
- Accepting constructive feedback gracefully.
- Focusing on what is best for the community and project.
- Showing empathy toward other contributors.
- Helping new contributors learn and improve.
- Collaborating in a respectful and constructive manner.

Examples of unacceptable behavior include:

- Harassment, discrimination, or hateful language.
- Personal attacks or insulting comments.
- Trolling or intentionally disruptive behavior.
- Publishing someone else's private information without permission.
- Threats, intimidation, or abusive conduct.
- Sexualized language or imagery.
- Spam, advertising, or unrelated promotional content.
- Any behavior that could reasonably be considered inappropriate in a professional environment.

---

# Project Responsibilities

Project maintainers are responsible for:

- Clarifying and enforcing this Code of Conduct.
- Reviewing reports fairly and objectively.
- Taking appropriate action when unacceptable behavior occurs.
- Creating a welcoming environment for contributors.
- Making moderation decisions that protect the community and project.

Maintainers have the right and responsibility to remove, edit, or reject contributions, comments, commits, issues, discussions, pull requests, or other content that violates this Code of Conduct.

---

# Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub Issues
- Pull Requests
- GitHub Discussions
- Project Wiki
- Documentation
- Code Reviews
- Community Chats
- Social Media interactions representing the project
- Any official Hyper Learning Tech community spaces

It also applies when an individual is officially representing the project in public spaces.

---

# Reporting Violations

If you experience or witness unacceptable behavior, please report it privately.

**Contact:**

**Email:** [hyperlearningtech@gmail.com](mailto:hyperlearningtech@gmail.com)

Please include:

- A description of the incident.
- Links or screenshots (if applicable).
- Any relevant context.

All reports will be reviewed promptly and handled confidentially to the extent possible.

---

# Enforcement Guidelines

Project maintainers may take actions including:

- Friendly warning.
- Request to modify behavior.
- Removal of inappropriate content.
- Temporary suspension from the project.
- Permanent ban from the community for repeated or severe violations.

Enforcement decisions will be based on the severity and impact of the behavior.
